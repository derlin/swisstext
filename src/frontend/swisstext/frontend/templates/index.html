Welcome to the SwissText Frontend!

This is the user interface to the SwissText - Swiss German Corpus Generator system. From here, you can monitor the system, validate and label sentences, export a dataset and more.

With SwissText, we pursue a semi-automatic human-in-the-loop approach for obtaining Swiss German sentences using guided web crawling and for performing dialect annotation.
