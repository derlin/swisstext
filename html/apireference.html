API Reference¶

The swisstext namespace contains the following:

  • Package mongo: abstract definitions of the MongoDB structure

  • Package cmd: impementation of two distinct commandline tools:

    • searching: discover new URLs from seeds using a Search Engine,

    • scraping: scrape the web and discover new Swiss German sentences.

  • Package frontend: a Flask application for visualisation and labelling
